# User Personas

## Overview

InsightOS serves diverse user groups with different needs, skills, and workflows. Understanding these personas guides product decisions and feature prioritization.

## Persona 1: Data Analyst - Sarah

### Profile
- **Age**: 32
- **Experience**: 8 years in data analysis
- **Technical Skill**: High (SQL, Python, Power BI)
- **Role**: Senior Analyst at mid-size retailer
- **Pain Points**: Manual data prep takes 40% of time, switching between tools

### Needs
- Automate data cleaning
- Generate insights faster
- Share analysis with team
- Maintain data quality standards

### Motivation
- Recognition for insights
- Career advancement
- Work-life balance

### Jobs to Be Done
- Profile new datasets
- Identify data issues
- Create executive summaries
- Collaborate with stakeholders

### Success Metrics
- 50% reduction in data prep time
- 10+ insights per week
- Team adoption of platform
